Addon Instructions

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://azndoodle.github.io/kodi_repo/repo Give it the name "azndumpster-repo".
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "azndumpster-repo", and install the latest repository.dumpster-x.xx.xxx.zip version.
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "Azn Dumpster Repository"
  • Go into the "Video add-ons" section in the repo, and you'll find the Icdrama plugin.
